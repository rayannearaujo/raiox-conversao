# Raio-X de Conversion

An AI-powered website conversion analyzer designed to identify opportunities for improving conversion, offer clarity, trust, visibility, and performance.

The platform evaluates strategic elements of a website and generates actionable insights to help businesses increase leads, sales, and overall digital performance.

## Features

* Conversion Rate Optimization (CRO) Analysis
* Offer Clarity Assessment
* Trust Signal Evaluation
* Visibility Analysis
* Performance Analysis
* Automatic Website Type Detection
* Lead Capture Integration
* Calendly Scheduling Integration
* Microsoft Clarity Integration
* Built with Next.js and TypeScript

## Tech Stack

* Next.js
* TypeScript
* Tailwind CSS
* Google PageSpeed Insights API
* Resend
* Microsoft Clarity
* Vercel

## Purpose

To help businesses quickly identify the factors that may be limiting their online conversions, lead generation, and growth potential.
